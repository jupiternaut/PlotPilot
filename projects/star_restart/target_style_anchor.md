# 《星河重启》目标样稿风格锚点

来源：`/Users/gengrf/Desktop/工作区/claude/小说/星河重启-首富穿越计划/10-正文前三章正式稿.md`

这个文件不是正文，不替代样稿本身。它用于让 PLM 在生成后续章节时保持前三章正式稿的口味。

## 结构锚点

- 第 1 章不是单纯世界观介绍，而是一条完整死亡链：海报 -> 买馒头 -> 电视竞选 -> AI 安全讽刺 -> 逗宝账单 -> 星网火星广告 -> 牛思克家族口角 -> 无人车撞死 -> 人类必胜 -> 回到夏天。
- 第 2 章不是重生惊喜，而是资产清点：高淳、十八岁、寄人篱下、朱姨、楼道议论、二十三块钱、夜场卖唱、Dota1 陪玩。
- 第 3 章不是开金手指，而是回到人间：十块钱、邻居闲话、旧会员卡、童年雨夜、砚山书局、沈砚山的面。

## 语言锚点

- 每个荒诞设定都要有收费细节、用户协议感或平台提示感。
- 每个温情场景都要有一个不煽情的物件：热糖水、包子、旧作文书、搪瓷碗、荷包蛋、台灯。
- 顾临川的内心不能像演讲。他会开玩笑，会算钱，会忍住反击，会把自己难堪的处境说成段子。
- 小镇人物的刺不靠恶毒台词，而靠熟练的家常话。
- 段落要有呼吸。不要一段里塞完所有解释。

## 续写硬约束

- 后续章节必须承认前三章已发生，不要重写穿越起点。
- 书店老板统一叫沈砚山，不使用旧占位名。
- 顾临川寄住在朱姨家。
- 过去地点以高淳为核心。
- 第一章标题是“低价值碰撞样本”，第二章“高淳的热”，第三章“寄人篱下”。
- 第 4 章以后再正式推进沈知夏。
